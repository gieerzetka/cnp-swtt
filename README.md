# Star Wars Top Trumps

## Instalation 

1. Clone repo: `git clone https://github.com/gieerzetka/cnp-swtt.git <YOUR_PROJECT_NAME>`
2. Install dependencies: `yarn install` or `npm install`

## Usage
### Development mode

With yarn:
```sh
yarn start
```
With npm:
```sh
npm start
```

### Production mode

With yarn:
```sh
yarn build
```
With npm:
```sh
npm run-script build
```

### Test mode

With yarn:
```sh
yarn test
```
With npm:
```sh
npm test
```
